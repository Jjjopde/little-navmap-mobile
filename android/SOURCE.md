# Corresponding source

Little Navmap Mobile is free software licensed under GPL-3.0-or-later.

The source for this unofficial Android adaptation, including its build
scripts, is the [`android/`](./) directory in the source package supplied with
the APK. The adaptation was initially based on Little Navmap revision
`bb2ac07934fcda6ccdf5b246394903e78e7a32ce`.

The original Little Navmap source is available from
[github.com/albar965/littlenavmap](https://github.com/albar965/littlenavmap).
That upstream repository might not contain this unofficial Android adaptation.
To obtain the exact Corresponding Source for a particular APK, use the matching
source package supplied by its distributor. If it is missing, request it from
the person or site that supplied the APK.

Anyone redistributing the APK must provide its complete corresponding source,
or another source-access method permitted by section 6 of GPLv3. The source
must match the APK, including Android modifications and build scripts.
